JavaScript SDK || Introduction & Configuration

The @Pay JavaScript SDK allows you to safely register @Pay accounts, tokenize Credit Card records and run Transactions.

Supported Browsers

In order to accept payments or register cards successfully with the @Pay JavaScript SDK, your Customer must be using IE8+, Chrome, Firefox 3+, or Safari 5.1+ (OSX / iOS) and must have Javascript enabled.

Installation

The SDK is loaded by placing the script element at the end of the body element. (Please load jQuery (>= 1.9.1) independently of the SDK.)

1
2
  <script src="//ajax.googleapis.com/ajax/libs/jquery/1.9.1/jquery.min.js"></script>
  <script type="text/javascript" src="https://dashboard.atpay.com/api/v4/atpay.js"></script>

            $(function( ){

                    test_mode:                  false
                  } );
            });

Attribute Description
organization_sid The Organization SID found in your Orgnaization Dashboard under 'API Settings'
default_gatway The SID of the Gateway that you want to assign to all Offers to (optional)
test_mode Creates all Invoices and Offers with a test Gateway. Will take precedence over any other gateway configuration. (optional)
