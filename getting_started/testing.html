Test Gateways & Cards

Every organization account comes with a test Gateway and a collection of test cards that can simulate transactions.

Test Cards

You can use the following test cards to simulate transactions on offers and invoices configured with test gateways.

Type Card Number Declined/Failed Card
Visa 4111 1111 1111 1111 4012 8888 8888 1881

Discover 6011 1111 1111 1117 6011 0009 9013 9424
American Express 3782 8224 6310 005 3714 4963 5398 431

Test Gateways

If you want to simulate a Test transaction with an offer, you must choose Test Gateway in the Offer Generator. If you are generating an offer from the JavaScript or another @Pay wrapper, you must set the offer_restricted_gateway argument to your Test Gateway SID. You can find the SID under "Manage Gateways" in your dashboard.
