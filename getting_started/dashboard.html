Your Organization's Dashboard

Every organization is provided with dashboard access. Here you can track your incoming transactions, create @Pay Offers and Buttons, adjust receipt settings and and configure API access. You can access your organization's dashboard at dashboard.atpay.com.

Dashboard Users

Once logged in, you can create additional User accounts to access various tools in your Organization's Dashboard. There are three "user types" each with different permissions. These include 'Owner', 'Marketer', and 'Developer'.
